# Examen Final - Serie II

Clase Alumno en TypeScript con gestión de lista de estudiantes.

## Ejecutar

```bash
npm start
# o
npm run ejercicio
```

## Funcionalidades

- Clase Alumno con 3 atributos (nombre, edad, carrera)
- Métodos para crear y obtener alumno
- Lista de 10 alumnos generados con bucle
- Ordenamiento por nombre
- Eliminación del primer y último registro
